🎯 **Live Platform**: [https://purpledroid.com](https://purpledroid.com)

---

## 🚀 Features
* **Multi-level Challenges**: From beginner-friendly web tasks to advanced binary reversing.
* **Realistic Environment**: Practice real-world exploit techniques safely.
* **Interactive UI**: Sleek frontend dashboard to track your CTF campaign progress.

* **Frontend**: Vite + JavaScript / CSS
* **Backend**: Python (FastAPI / Uvicorn)
* **Web Server**: Caddy (Automated SSL)
* **Target Environment**: Android & Linux Binaries
